# My Personal Website

My Personal Website, website :D

## Setup

Install Jekyll and NPM modules list in package.json:

```bash
$ gem install jekyll --no-document
$ npm install
```

### Working

To run jekyll server:

```bash
$ jekyll server
```

And to waatch for file changes in stylesheets:

```bash
$ npm start
```

### Releasing

Before releasing, make a build of the files:

```bash
$ npm run build
```

Commit and push to master.
