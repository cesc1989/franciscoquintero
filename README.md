# My Personal Website

My Personal Website, website :D

## Setup

Install NPM modules list in package.json:

```bash
$ npm install
```

### Working

To watch for file changes in stylesheets:

```bash
$ npm start
```

### Releasing

Before releasing, make a build of the files:

```bash
$ npm run build
```

Commit and push to master.
